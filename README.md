# Проект верстка сайта для произлдителей кофе
## Проект
Этот репозиторий является каркасом для будущего веб-приложения.

Для css я выбрала методологию БЭМ для обеспечения четкой структуры компонентов, уникальности имен классов. Эта система удобна для работы,в ней понятная система именования, что упрощает понимание кода.
